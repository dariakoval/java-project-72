# Page analyzer

Page Analyzer is a website that analyzes certain pages for their suitability for SEO.

This is an educational project based on the Javalin framework.

## Requirements

* JDK 20
* Gradle 8.2
* GNU Make

## Setup

```bash
make setup
```

## Run server

```bash
make start
# Open http://localhost:7070
```
